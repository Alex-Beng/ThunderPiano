# ThunderPiano --- fast like Tunder

面向手游《别踩白块》。拟使用scrcpy的服务端进行屏幕串流及屏幕触控。编写客户端使用计算机视觉技术进行识别并与sever通信完成点击。

目标为完成一个实时，非Hack方式，速度媲美正常人类的手游《别踩白块》纯软件jio本。
